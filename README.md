# JSTQB AL テストアナリスト 学習アプリ

JSTQB Advanced Level テストアナリスト（CTAL-TA）資格対策の個人学習用Webアプリ。
シラバス **Version 3.1.1.J03**（6章構成）に準拠。

## 特徴

- **模擬試験モード** … 全問をランダム出題。配点ベースで採点（合格ライン65%）。
- **章別演習** … 6章ごとに絞って演習。
- **復習モード** … 間違えた問題だけを再出題（正解すると対象から外れる）。
- **解説** … 各問に正誤理由とシラバス該当箇所(LO/知識レベル)を表示。
- **進捗管理** … 学習率・正答率・章別進捗を可視化。端末内(localStorage)に保存。
- **要点まとめ** … 章ごとのシラバス要約を閲覧。

## 技術構成

PhysiqueLog と同方式。ビルド工程なしのバニラ構成。

| ファイル | 役割 |
|---|---|
| `index.html` | アプリ本体（HTML/CSS/JS、画面遷移・採点ロジック） |
| `data.js` | 問題バンク＋章メタ情報＋要点まとめ |

- データ保存: `localStorage`（キー `jstqb_ta_v1`）。サーバー不要。
- スマホ型レイアウト（最大幅540px）。PC・スマホのブラウザ両対応。

## 使い方

`index.html` をブラウザで開くだけ。

会社内へのクローズ公開は以下のいずれかで:
- Cloudflare Pages 等の静的ホスティングにアップロードしてURL共有
- Android WebView ラッパー（PhysiqueLog と同方式）でAPK配布

## 問題の追加

`data.js` の `QUESTIONS` 配列に追記する。

```js
{
  id: "q-3-07",            // 一意なID
  chapter: 3,              // 章番号(1-6)
  lo: "TA-3.2.6",          // 学習目標
  k: "K3",                 // 知識レベル K2/K3/K4
  points: 2,               // 配点 1/2/3
  type: "single",          // "single"=1つ選ぶ / "multiple"=複数選ぶ
  scenario: "（任意）長文シナリオ",
  question: "設問文",
  options: [ {id:"a",text:"..."}, {id:"b",text:"..."} ],
  correct: ["a"],          // 正解の選択肢id（複数可）
  explanation: "解説文"
}
```

## 注意

- 問題はISTQBサンプル試験 Set A を題材に日本語で作成。逐語転載はしていない（個人学習用、再頒布しない）。
- 商用配布する場合は ISTQB/JSTQB の著作権・利用条件を別途確認すること。
- 最新版シラバス **テストアナリスト V4.0.J01**（2026年6月公開）あり。本アプリはV3.1.1準拠。
